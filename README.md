# Java Calculator

## Description
This is a simple calculator application developed using Java. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an easy-to-use interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Simple and user-friendly design

## Technologies Used
- Java
- Java Swing (or Java AWT, if you used it)

## How to Run
1. Clone or download the project.
2. Open it in your preferred Java IDE (Eclipse, IntelliJ IDEA, or NetBeans).
3. Compile and run the `Calculator.java` file.

## Learning Outcomes
- Understanding Java programming concepts
- Implementing arithmetic operations
- Building a basic graphical user interface
- Handling user input and events
